# Data Science Lab - Data Preprocessing Using Python

## Overview

This repository contains **Lab Exercise I: Data Preprocessing Using Jupyter Notebook (Google Colab)**. The objective of this lab is to perform various data preprocessing techniques on the **Heart Disease Dataset** using Python libraries such as Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.

---

## Dataset

- **Dataset:** Heart Disease Dataset
- **Source:** Kaggle
- **Records:** 1025
- **Attributes:** 14

---

## Tools & Libraries

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Preprocessing Tasks Performed

1. Loaded the dataset using Pandas.
2. Displayed basic information about the dataset.
3. Identified rows, columns, data types, and missing values.
4. Performed exploratory data inspection.
5. Detected and handled missing values.
6. Identified and removed duplicate records.
7. Detected outliers using the IQR method.
8. Handled outliers using IQR capping.
9. Converted categorical variables into numerical form.
10. Applied Min-Max Normalization.
11. Applied Standardization (Z-score Scaling).
12. Performed feature selection using correlation analysis.
13. Generated a correlation matrix and heatmap.
14. Performed feature engineering.
15. Converted data types where necessary.
16. Cleaned inconsistent and noisy data.
17. Split the dataset into training and testing sets.
18. Visualized feature distributions before and after preprocessing.
19. Compared the dataset before and after preprocessing.
20. Saved the cleaned dataset as a new CSV file.
